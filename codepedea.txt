Família Inline(Em linha) e Block(em bloco)

    Essa divisão influencia diretamente a
    forma como os elementos são exibidos e
    organizados na estrutura da página.

    Elementos block ocupam toda a
    largura disponível e sempre
    começam em uma nova linha. 

    Elementos inline ocupam apenas o
    espaço necessário ao seu conteúdo.

{Display}

    block: Ocupa toda a linha. Permite
    largura, altura, margem e padding.


    inline: Fica na mesma linha dos outros.
    Não aceita largura/altura.


    inline-block: Fica em linha, mas aceita
    largura e altura.


    none: Esconde o elemento da página
    (como se não existisse).


    flex: Organiza os elementos filhos em
    linhas ou colunas com layout flexível.


    grid: Cria layouts em grade (linhas e
    colunas).

Tamanho

    width: define a largura do elemento.
    height: define a altura do elemento.


    px (pixels – valor fixo)
    % (porcentagem do elemento pai)
    vw/vh (largura/altura da tela)
    em/rem (relativas ao tamanho da fonte)


    Só funcionam em elementos com display:
    block, inline-block, flex, grid etc.


    vw (viewport width): representa 1% da largura da tela
    (janela do navegador).

    Ex: width: 50vw; → ocupa 50% da largura da tela.


    em: unidade relativa ao tamanho
    da fonte do elemento pai.
    Ex: se o pai tem font-size:
    16px;, então 1em = 16px.


    rem (root em): relativa ao tamanho
    da fonte do elemento raiz (html).
    Ex: se o html tem font-size: 16px;,
    então 1rem = 16px, sempre.

Margem e espaçamento

    padding: Espaço interno do elemento (entre o
    conteúdo e a borda).


    margin: Espaço externo do elemento (afasta
    dos outros ao redor).


    Ordem: Topo  direita  baixo  esquerda. 

    Ordem 2: Topo/Baixo Direita/Esquerda.


    auto significa que o navegador vai calcular o
    valor automaticamente, com base no contexto.

    auto nas laterais → centraliza o elemento se
    ele tiver largura definida;

    Em height: auto; ou width: auto;, o navegador
    calcula o tamanho com base no conteúdo.

Class e ID

    As atribuições class e id servem para identificar
    elementos HTML e permitir que sejam estilizados com
    CSS ou manipulados com JavaScript.

    class: usada para agrupar vários
    elementos com o mesmo estilo.


    id: usada para identificar um único
    elemento de forma única na página.

{FlexBox}

Flexbox é um modelo de layout que facilita a organização
de elementos em linha ou coluna, adaptando-se a
diferentes tamanhos de tela.

 flex-direction: define a direção

row (padrão), column, row-reverse, column-reverse


justify-content: alinha os itens no eixo principal

flex-start, center, space-between, space-around,
space-evenly


 align-items: alinha os itens no eixo cruzado

stretch (padrão), center, flex-start, flex-end


flex-wrap: define se os itens quebram para novas linhas

 nowrap (padrão), wrap, wrap-reverse

{FlexBox Itens}

Order: Define a ordem de exibição dos itens.


Flex-Grow:  Controla quanto o item pode crescer para
ocupar o espaço livre.


Flex-Shrink: Controla quanto o item pode encolher quando
não há espaço suficiente.


flex-basis: Define o tamanho inicial do item antes de
distribuir o espaço.


Gap: Define o espaço entre os itens dentro do container
flex.


Align-Self: Permite alinhar individualmente um item na
vertical.
